# Vue TS + Storybook

Vite + Vue + TypeScript + Storybook 製の GitHub Pages です。

- [GitHub Pages - Vue TS + Storybook](https://mkrydik.github.io/vue-ts-storybook)


## コーディング関連コマンド

```bash
$ npm install

# Vite
$ npm run dev
$ npm run build
$ npm run serve

# Storybook
$ npm run dev-storybook
$ npm run build-storybook
$ npm run serve-storybook
```
